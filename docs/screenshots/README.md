# Screenshots

Drop the screenshots referenced by the main `README.md` into this folder, using
these exact filenames so the images render automatically:

| Filename             | What to capture                                                        |
|----------------------|------------------------------------------------------------------------|
| `menubar.png`        | The DockSnap icon in the macOS menu bar.                               |
| `automatic-tab.png`  | The popover open on the **Automatic** tab (modifier picker + toggles). |
| `manual-tab.png`     | The popover open on the **Manual** tab with a couple of custom rows.   |
| `accessibility.png`  | System Settings → Privacy & Security → Accessibility with DockSnap on. |

Tip: press <kbd>⌘⇧4</kbd> then <kbd>Space</kbd> to capture a single window with a
clean drop shadow. Keep images under ~1 MB (PNG is fine).
